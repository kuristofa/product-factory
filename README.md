# AI-powered Product Production Factory

Turn an approved offer into a complete, ready-to-sell product package — the product,
the content, the DM flow, and the checkout copy — all compiled into one PDF.
You review at two points; the AI does the rest.

---

## Set it up (once per computer)

You only do this one time.

1. Double-click **`Setup.bat`** and follow the on-screen messages.
2. Install the AI engine and sign in:
   - Install **Node.js** (the LTS version) from https://nodejs.org
   - Open a terminal and run: `npm install -g @anthropic-ai/claude-code`
   - Then run: `claude` and sign in with your **Claude Max** account.

That's it — you're done.

---

## Use it (every day)

1. Double-click **`Start.bat`** — the app window opens.
2. Pick a saved brief, **or drag in the offer file** from your Grand Slam Offer GPT
   (a PDF, `.md`, or `.txt`).
3. Click **Start** and watch it work.
4. It pauses **twice** for you: a quality review, then a final sign-off. Read, then Approve.
5. **Download your PDF.** Done.

> New to it? There's a `READ ME FIRST.txt` in the folder with the same steps in plain language.

---

## Good to know

- It runs the AI on **this** computer using the Claude account it was set up with — so it has
  to run here.
- It never makes up the offer. It builds from the offer you give it.
- Nothing is published automatically. You approve everything.
- If the app says setup isn't finished, tell whoever set it up.
